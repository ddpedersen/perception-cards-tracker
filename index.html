🔑 Connect to GitHub

This tracker stores your data in GitHub. You'll need a Personal Access Token — create one once, enter it here, and it's saved to this browser.

  1. Go to github.com → Settings → Developer settings → Fine-grained tokens
  2. Click Generate new token
  3. Name it anything (e.g. perception-cards)
  4. Under Repository access → Only select repositories → pick perception-cards-tracker
  5. Under Permissions → Contents → set to Read and write
  6. Click Generate token and copy it
or migrate existing data

Already have a backup JSON from the old tracker? Import it after connecting.
